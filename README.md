# My MenoPower — AI Agent Lab 🤖

Portfolio AI Agent Managera budowane na marce wellness **My MenoPower**.

## 🎯 O projekcie

Przez 90 dni uczę się budować systemy AI agentów — jednocześnie automatyzując
procesy biznesowe własnej marki wellness dla kobiet w perimenopauzie.
Każdy projekt rozwiązuje realny problem My MenoPower.

## 🗺️ 5 projektów w budowie

| # | Projekt | Co robi | Stack |
|---|---------|---------|-------|
| 1 | **MenoGuide AI** | Chatbot odpowiadający na pytania klientek | LangChain + RAG + Streamlit |
| 2 | **Content Engine** | 5 agentów tworzy artykuły + SEO + social | CrewAI + Tavily |
| 3 | **Lead Agent** | Automatyczny onboarding nowych leadów | n8n + Airtable + Gmail |
| 4 | **Analytics Agent** | Tygodniowe raporty trendów i pytań | Python + OpenAI |
| 5 | **Expert Copilot** | AI asystent dla wellness coacha | Streamlit + OpenAI |
